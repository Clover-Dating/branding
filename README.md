# Clover Branding

Brand assets for Clover Dating.

## Icons

Pixel art icons in `icons/`. All icons available at 1x (12px), @2x (24px), @3x (36px).

**Available:** account, clover, developer, edit, heart, heart-alt, heart-outline, hide, home, mailbox, messages, moon, notifications, profile, question, report, rocket, search, settings, signout, star, star-alt1, star-alt2, sun, sun-alt1, sun-alt2, visitors

Use `tintColor` to colorize dynamically instead of separate light/dark files.

## Logomarks & Favicons

- `logomark.png` / `logomark-1x.png` - Clover icon
- `favicon.ico` / `favicon-{16,32,64,128,256}.png` - Browser favicons

## Social / Marketing

- `avatar.png` (400x400) - Profile avatar
- `avatar-logomark.png` - White logomark on green
- `banner-grass.png` (1500x500) - Twitter banner (grass pattern)
- `banner-logomark.png` - Logomark banner
- `banner-logo.png` / `banner-cloverdating.png` - Logo banners

## Brand Colors

- **Clover Green:** `#62a94c`

- **Dark bg:** `#1A1612`

- **Dark text:** `#CDC8C2`
