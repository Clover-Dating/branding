# Clover Branding

Brand assets for Clover Dating.

## Icons

Pixel art icons in `icons/`. Icons available at 1x (12px), @2x (24px), @3x (36px).

Use `tintColor` to colorize dynamically instead of separate light/dark files.

## Logomarks & Favicons

There are a variety of logos, wordmarks, logomarks, avatars, and favicons.
